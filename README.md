## Main commands

Install the dependencies:

```
npm install
```

Start a local MongoDB server:

```
mongod
```

Develop locally: 

```
npm start
```

Test locally:

```
npm test
```

Deploy to staging:

```
npm run deploy
```

Deploy to production:

```
npm run deploy:prod
```
